## Techs Used
- Django

# To Setup Project In Local
## Create python env
- python3 -m venv .venv
- activate : source .venv/bin/activate

## install requirements file
- pip install -r requirements.txt

## setup tailwind
- npm init y
- npm install 

## run the code 
- python manage.py runserver
